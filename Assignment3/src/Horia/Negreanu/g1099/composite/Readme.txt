Task Management application for businesses:

There is a department of project managers, which add tasks, and this department of project managers has another department
of programmers below in hierarchy which receive the tasks in order to complete them.

Advantages:
1. Allows departments to contain both just employees, and other subordinate departments.
2. Leaf node (simple employees) and departments are treated the same in the structure.

Disadvantages:
1. Harder to implement
